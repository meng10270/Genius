# 项目介绍
## 项目内容
    主要完成网上书城购物的用户注册、用户登陆、主页展示、购物车、商品详情、商品评价、下订单与支付、个人信息管理、订单管理等功能。
## 开发工具：NetBeanS、MySQL、GitHub
